# Data Analytics Project; Customer Behavior Analysis: End-to-End Business Insights

 ## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating business insights. The objective is to analyze customer and sales data to uncover trends, improve decision-making, and present insights through an interactive dashboard.

 ## Objectives
Perform data cleaning and preprocessing
Conduct Exploratory Data Analysis (EDA)
Extract insights using SQL queries
Build an interactive Power BI dashboard
Generate a business-focused analytical report

## Tools & Technologies
Python (Pandas)
MySQL (Data querying & analysis)
Power BI (Dashboard & visualization)


## Project Workflow

1. Data Loading & Exploration (Python)
Imported dataset using Pandas
Checked data structure, types, and summary statistics
Identified missing values and inconsistencies

3. Data Cleaning
Handled missing values and duplicates
Standardized column names and formats
Corrected data types for analysis

5. Exploratory Data Analysis (EDA)
Analyzed customer behavior and purchasing patterns
Identified key trends across categories (e.g., age group, gender, product type)
Visualized insights using charts and graphs

7. SQL Analysis (MySQL)
Performed aggregation and filtering using SQL
Identified top-performing products and customer segments
Calculated KPIs such as average rating, discount rate, and repeat customers

9. Power BI Dashboard
Designed an interactive dashboard for business insights
Included KPIs, charts, and filters for dynamic analysis
Focused on usability and storytelling

11. Reporting
Summarized key findings and business recommendations
Highlighted actionable insights for decision-making

  ## Key Insights
Majority of customers are not subscribed → opportunity for targeted offers
Certain customer segments (e.g., young adults) contribute higher spending
Popular shipping methods indicate preference for speed and convenience
Opportunities exist to improve engagement in underperforming segments
